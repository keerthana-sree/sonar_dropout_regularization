# Dropout Regularization using Artificial Neural Network (ANN)

This project demonstrates the implementation of Dropout Regularization in Artificial Neural Networks (ANN) using Deep Learning techniques.  
The model is developed to reduce overfitting and improve the performance of neural networks.

---

## 📌 Project Overview

The main objective of this project is to understand and implement Dropout Regularization in ANN models for better generalization and prediction performance.

The project includes:
- Data preprocessing
- ANN model creation
- Dropout layer implementation
- Model training and testing
- Accuracy evaluation
